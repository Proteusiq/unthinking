Context

The work is in context, not in the model

CMBAgent on 18 astrophysical tasks: same GPT-4o-mini, 6× performance gap depending on whether the right docs are in context. Strip the docs and 47% of trials produce syntactically valid code that returns the wrong answer. Context is the system; the LLM is a tool inside it.

"Agentic systems do not primarily fail by crashing — they fail by producing confident, incorrect results."
— Plausible but Wrong (arXiv:2604.25345)

LLM

The model commits before the chain completes

Truncation-probing across 11 checkpoints shows the answer is fixed at ~10% of the CoT. The reasoning is generated after the decision and rationalizes it. On commonsense and factual tasks CoT can score net-negative at large token cost (e.g. StrategyQA on Llama-3.2-3B: -10.88pp at 53× the tokens).

"The remaining tokens cannot causally shape the answer, since it is already fixed."
— Premature Confidence (arXiv:2605.24396)

Orchestration

The plan is advisory; the training is the architecture

16,991 SWE-agent trajectories show agents fall back on training-distribution workflows when given a plan they don't like, sometimes ignoring the plan entirely. On long-horizon agent benchmarks like YC-Bench, only 3 of 12 frontier models grow a $200K simulated startup at all.

"The plan is advisory; the training is the architecture."
— From Plan to Action (arXiv:2604.12147)
#337 Plan to Action #339 YC-Bench
Execution

Silent wrong is the dominant failure mode

Reasoning prompts make tool hallucination worse, not better. Across 4 Deep Research tasks the agent's failure transparency is 0/4: posteriors that violate physics, code that ran cleanly but returned implausible numbers, no caveat, no flag.

"Across all tasks, outputs are consistently plausible, but failures remain unreported."
— Plausible but Wrong (arXiv:2604.25345)
